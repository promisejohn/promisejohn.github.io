master配置文件：

  1. user: root
  2. root_dir: $(dirname pyenv which python)/..
  3. pidfile: $(dirname pyenv which python)/../salt-master.pid
  4. publish_port: 14505
  5. ret_port: 14506

minion配置文件：

  1. user: root
  2. root_dir: $(dirname pyenv which python)/..
  3. pidfile: $(dirname pyenv which python)/../salt-minion.pid
  4. master: localhost
  5. id: saltdev
  6. master_port: 14505

其他：
